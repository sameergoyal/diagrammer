[Diagrammer](https://github.com/sameergoyal/diagrammer) is a library to display an interactive editor for any graph-like data.

Take a look at how one of the consumers of this library, [AWS IoT Events Console](https://console.aws.amazon.com/iotevents/home?region=us-east-1#/create/detectormodel) uses Diagrammer in their product.

# Why Diagrammer?

Diagrammer is a framework & data format agnostic library that is fully customizable in terms of look & feel as well as behavior. It also exposes a declarative interface to reduce the code required to integrate the library in any application and comes with many interactive features built in. Read more about the features of the library [here](?path=/story/docs-features--page).
